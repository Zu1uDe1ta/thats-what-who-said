# OTHER INFO

Two CSV files containing quotes from the 4 main characters of The Office: Michael, Dwight, Jim and Pam.

- talking_head.csv contains a character's monologue when talking directly to the camera.

- parent_reply.csv contains a character's quote when replying to another character's line.
